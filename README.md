# Student Grade Predictor 🎓

A Machine Learning project that predicts whether a student will Pass or Fail based on their attendance, midterm marks and assignment marks.

## About the Project
This project uses Logistic Regression algorithm to predict student results. Teachers can use this tool to identify weak students early and help them improve.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Google Colab

## How it Works
1. Student attendance, midterm and assignment marks are given as input
2. Logistic Regression model analyzes the data
3. Model predicts Pass or Fail as output

## Input Features
- Attendance (percentage)
- Midterm Marks (out of 100)
- Assignment Marks (out of 100)

## Results
- Total Students: 20
- Model trained successfully
- Pass and Fail correctly predicted ✅
